# Calorie Counter App

##  Project Description
The Calorie Counter App is a simple web application that helps users track their daily calorie intake. Users can add food items, view total calories, remove items, and reset their daily log. Data is saved using localStorage so it stays even after refreshing the page.

---

##  Features
- Add food items with calorie values
- Display all food items
- Remove individual items
- Calculate total calories automatically
- Reset all data for a new day
- Save data using localStorage
- (Optional) Fetch calorie data using async JavaScript (simulated API)

##  Technologies Used
- HTML5
- CSS3
- JavaScript 
- DOM Manipulation
- localStorage
- Fetch API 

##  Setup Instructions

Follow these steps to run the project locally:

 1. Clone the repository  ```https://github.com/tugiii45/Calorie_counter.git```
 2. Open the project folder
 3. Open in browser and you're in

 ## How to use the app

1. Open the application in your browser.
2. Enter a food name in the input field.
3. Enter the calorie value (or let API simulate it if implemented).
4. Click "Add Food".
5. The food item will appear in the list below.
6. The total calories will update automatically.
7. Click "X" to remove a food item.
8. Click "Reset" to clear all data.
